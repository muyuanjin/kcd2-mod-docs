# Detail maps - Library chart
You will find here a full detailing of the detail map library documented with names, pictures and the grayscale value matching each detail map.

The structure of the library is quite simple, texture name and the **grey scale value**, these are the value **in rgb to set up the correct grayscale shade that matches the detail map**. Ex: Photoshop: 8,8,8
![](image1.png)
**Or by using the material channel slider.** ex: Painter: 0.02 to 0.04
![](image2.png)

Each one will be accompanied by screenshots of how the detail looks applied in the engine with the corresponding tilling value and their actual look as .tif file

**!!! some maps are oriented Horizontally or Vertically, theses are marked at the end with "h" or "v"!!!**
