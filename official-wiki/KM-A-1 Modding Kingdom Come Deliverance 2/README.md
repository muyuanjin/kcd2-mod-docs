# Modding Kingdom Come: Deliverance 2
This wiki is a documentation for modding for the game Kingdom Come: Deliverance 2.

* If you are a consumer of mods looking for how to install mods, see [KM-A-56](../KM-A-56 Installing mods/README.md)
* If you are a modder, start by looking at [KM-A-36](KM-A-36 Technical Overview/README.md)

This youtrack server is not intended for customer support or reporting bugs with the actual game. For that use [Kingdom Come: Deliverance II Player Support](https://support.plaion.com/en/games/kingdomcome2/)



<span style="background-color:mistyrose;">Comments policy</span>: Comments on this wiki are enabled; however, they are not intended to be used as a public forum or customer support, primarily due to the technical limitations of this platform. Please use comments as notes to article authors - e.g., when important information is missing from the documentation. Such comments will be incorporated, the missing information added to the article, and then the comment will be deleted to avoid clutter. Any other comments (requests for help with user-specific problems, personal opinions, questions unrelated to modding, etc.) will be deleted outright.
